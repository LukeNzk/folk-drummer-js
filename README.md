# Traditional Drum Machine

Play here https://bebnista.pl/

Based on archival recordings, great for practicing polish traditional music with correct pulse.

This is 3rd attempt first one I did was React, then Vue.js. Vue was significantly faster than React, but still not snappy enough and overcomplicated. Then I decided to redesigne the GUI and make everything in vanilla js, works great, have fun!
